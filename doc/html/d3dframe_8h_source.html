00020 // Desc: Class to manage the Direct3D environment objects such as buffers,
00021 //       viewports, and 3D devices.
00022 //
00023 //       The class is initialized with the Initialize() function, after which
00024 //       the Get????() functions can be used to access the objects needed for
00025 //       rendering. If the device or display needs to be changed, the
00026 //       ChangeDevice() function can be called. If the display window is moved
00027 //       the changes need to be reported with the Move() function.
00028 //
00029 //       After rendering a frame, the ShowFrame() function filps or blits the
00030 //       backbuffer contents to the primary. If surfaces are lost, they can be
00031 //       restored with the RestoreSurfaces() function. Finally, if normal
00032 //       Windows output is needed, the FlipToGDISurface() provides a GDI
00033 //       surface to draw on.
